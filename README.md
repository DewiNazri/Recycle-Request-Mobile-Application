# ♻️ Recycle Request Mobile Application
Android mobile application for recycling pickup requests using REST API, developed with Java and integrated with backend services.

## ✨ Features

* 📦 Submit recycling pickup requests
* 📍 Track request status
* 🧾 Manage recyclable item details
* 🔐 User authentication (login/register)
* 🔄 Real-time data communication with backend API

---

## 🛠️ Tech Stack

**Frontend (Mobile App):**

* Java (Android Studio)

**Backend:**

* REST API (Prestige API / PHP / MySQL) 

**Tools & Technologies:**

* Android Studio
* Laragon 
* phpMyAdmin (database management)

---

## 🏗️ System Architecture

```
Mobile App (Android)
        │
        ▼
   REST API
        │
        ▼
   Database (MySQL)
```

---



## 🚀 How to Run the Project

### 📱 Mobile App

1. Open project in Android Studio
2. Sync Gradle
3. Run the app using emulator or physical device


## 🔗 API Integration

The application communicates with a backend REST API to:

* Submit recycling requests
* Retrieve user data
* Update request status
